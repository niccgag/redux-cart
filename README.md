# Redux Cart

A shopping cart application built with React and Redux Toolkit.

## Features

- Add items to cart
- Remove items from cart
- Adjust item quantities
- Clear cart functionality
- Modal confirmation dialogs
- Automatic total calculation

## Tech Stack

- React 17
- Redux Toolkit
- React Redux
- Vite

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy

```bash
npm run deploy
```
